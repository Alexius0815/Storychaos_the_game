# Story Chaos

Das Partyspiel gegen den Erzaehler.

## Deploy auf Vercel (5 Minuten)

1. Diesen Ordner als ZIP entpacken
2. Auf github.com: "New repository" → `story-chaos` → Dateien hochladen
3. Auf vercel.com: "Add New Project" → GitHub Repo auswaehlen → Deploy
4. Fertig! Deine URL lautet: `story-chaos.vercel.app`

## Als iPhone App speichern (PWA)

1. App-URL in Safari oeffnen
2. Teilen-Button (Kasten mit Pfeil nach oben)
3. "Zum Home-Bildschirm" tippen
4. Fertig — sieht aus wie eine echte App!

## Lokale Entwicklung

```bash
npm install
npm run dev
```
